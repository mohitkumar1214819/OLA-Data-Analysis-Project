# 🚖 OLA Data Analysis Project

## 📌 Overview

This project focuses on analyzing ride data from OLA to uncover business insights and improve decision-making. It covers the complete data analytics workflow, including data cleaning in Excel, querying using SQL, and building an interactive dashboard in Power BI. The objective is to identify trends in ride demand, revenue, and customer behavior.

---

## 📂 Dataset

* The dataset contains OLA ride-related data such as:

  * Booking details
  * Ride status (completed, cancelled, etc.)
  * Pickup and drop locations
  * Fare amount and payment type
* Data was preprocessed and cleaned using Excel before analysis.

---

## 🛠️ Tools & Technologies

* **Excel** – Data cleaning and preprocessing
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying and analysis
* **Power BI** – Dashboard creation and data visualization

---

## 🔍 Project Steps

### 1. Data Cleaning (Excel)

* Removed duplicates and handled missing values
* Standardized column formats (dates, text, numbers)
* Ensured data consistency for analysis

### 2. SQL Analysis

* Imported cleaned data into relational databases
* Performed SQL queries to analyze:

  * Total rides and revenue
  * Ride status distribution (completed, cancelled)
  * Peak booking times
  * Location-based demand
* Used aggregations, joins, and filtering for insights

### 3. Data Visualization (Power BI)

* Connected SQL/Excel data to Power BI
* Built an interactive dashboard with:

  * KPIs (total rides, revenue, cancellation rate)
  * Time-based trends
  * Location-wise analysis
* Enabled filters for dynamic exploration

### 4. Reporting

* Summarized key findings and insights
* Highlighted business trends and recommendations
* Created a structured report for stakeholders

---

## 📊 Dashboard

The Power BI dashboard includes:

* Key performance indicators (KPIs)
* Ride trends over time
* Revenue analysis
* Cancellation insights
* Location-based ride distribution


---

## 📈 Results & Insights

* Identified peak hours and high-demand locations
* Analyzed revenue patterns and ride distribution
* Highlighted cancellation trends and possible causes
* Provided actionable insights to improve operations and customer experience
